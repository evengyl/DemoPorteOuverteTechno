Quest-ce que Ionic?

Ionic Framework Est un SDK (service kit development)
Open source permettant aux développeurs de construire des application mobiles de haute qualités avec une technologies web comme Html, Css, Javascript

Ionic en deux mots

Ionic De la CI/CD (intégration continue) mobile aux micro-interfaces, Ionic vous aide à répondre à la demande de mobilité dans votre entreprise, tout en permettant aux équipes existantes de construire avec les outils qu'elles connaissent et apprécient.

4 façons de coder

Ionic Angular
Ionic React
Ionic Vue
Ionic JS (native)
Ionic va vous permettre de réaliser vos applications avec votre technologies front-end préférées !
